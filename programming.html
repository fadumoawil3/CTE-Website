SOFTWARE DEVELOPMENT PATHWAY

What Do We Offer?

Horizon Science Academy’s Software Development Pathway prepares students for dynamic careers in software engineering, web development, and information technology through hands-on coding projects, industry-recognized certifications, and real-world learning experiences. With an emphasis on programming, web technologies, and IT fundamentals, students acquire the technical expertise necessary to succeed in today’s rapidly evolving digital economy.

Classes

Students build a strong foundation in software development and IT through a carefully designed curriculum.

  • Information Technology
  • Programming
  • Object-Oriented Programming
  • Web Design
  • IT Capstone

Industry-Recognized Certifications

Graduates earn valued programming & IT certifications that enhance employability and prepare them for both college and direct entry into the tech workforce:

  • HTML Certification
  • CSS Certifications
  • Java Certification
  • JavaScript Certification
  • Python Certification
  • SQL Ceritification

Clubs

Students can enrich their learning and professional network through extracurricular activities such as:

  • Girls Who Code
  • Business Professionals of America (BPA)
  • Robotics
  • Drone Club
  • ESports

Careers

Graduates of the Software Development Pathway are well-prepared for high-demand roles including:

  • Software Developer
  • Web Developer
  • Mobile App Developer
  • IT Support Specialist
  • Cybersecurity Analyst
  • Game Developer
  • Cloud Computing Specialist
  • Data Analyst
